# Effect runtime visualizer

This is a visualizer for Effect runtime. It allows you to visualize the execution of Effect code in the browser.

This project is a learning project for me to understand the Effect runtime. Check out the [AGENTS.md](./AGENTS.md) file for more information:

- Most of the UI is generated by Cursor
- The Effect related code is written by me, guided by Cursor

## Scripts

- `npm run dev`: Start the development server
- `npm run build`: Build the project
- `npm run preview`: Preview the project
- `npm run lint`: Run ESLint
- `npm run lint:fix`: Fix ESLint errors
- `npm run test`: Run tests
- `npm run test:watch`: Run tests in watch mode
- `npm run typecheck`: Run type checking
